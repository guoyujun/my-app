### Target
> 使用React技术开发emp前端.

### Tech Stack
* UI库：React React-Dom
* 路由：React-Router
* JS：ES6
* 样式：css
* 打包构建：Babel Webpack
* 包管理：Npm

### Features
* 支持ES6
* React-Router配置路由
* 支持jQuery
* 支持页面自动刷新

### Usage
> 1、进入开发路径：$ cd public -> $ cd www -> $ cd h5
> 2、安装依赖：$ npm install  
> 3、启动服务：$ npm start  
> 4、standard校验：$ npm test
> 5、生成文件：$ npm run build
